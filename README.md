# Hacker News for iOS

__Hacker News API를 활용하여 REAST API를 연습하기위한 프로젝트입니다.__
- iOS version : 11.0

## 사용한 API
* Hacker News API => jobs 목록을 가져오기 위해사용
* HN Search API => 제일 기본적으로 사용한 API
